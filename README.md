What is WindowedClaude? -- >

Claude for windows today has a huge problem: for must, simple users, claude is almost impossible
to install and easily use on windows. It requires running multiple different installation commands
as well as dealing with Window's terrible 'power'shell to be utilized. This is where windowedclaude 
can step in!

WindowedClaude wraps every install process of claude into a single, easy to use runtime, creates 
a desktop shortcut for ease of access, and opens a friendly terminal for claude with clean themes 
and easy to use functions. WindowedClaude bypasses the utter complete annoyances of installing 
claude on windows into a straighforward and simplified process that anyone can understand.
